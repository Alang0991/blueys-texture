Custom VRChat and Unity texture shaders with emission support, glow effects, and custom material inspectors.

> Currently intended for **PC Only**.


---

## Installation

### Add Repository

Add this repository to VRChat Creator Companion:

```text
https://raw.githubusercontent.com/Alang0991/blueys-texture/main/index.json
```

### Install Through VCC

1. Open VRChat Creator Companion
2. Open your project
3. Click Manage Project
4. Open Packages
5. Install Blueys Texture

---

## Features

### Texture Controls

- PNG texture support
- Texture tinting
- Brightness controls
- Contrast controls
- Saturation controls

### Emission System

- Emission textures
- Emission masks
- Emission colour controls
- Adjustable emission intensity
- Main texture emission blending

### Visual Effects

- Rim glow
- Smoothness controls
- Metallic controls
- Colour overlay controls

### Material Inspector

- Custom Blueys material editor
- Organised foldout sections
- Cleaner workflow
- Easy-to-use controls
